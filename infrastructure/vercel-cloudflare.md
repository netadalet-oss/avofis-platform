# Vercel + Cloudflare

## Önerilen domainler
- `portal.avofis.com` → portal
- `admin.avofis.com` → admin
- `api.avofis.com` → api

## Portal environment
- `NEXT_PUBLIC_API_URL=https://api.avofis.com/api`

## Admin environment
- `NEXT_PUBLIC_API_URL=https://api.avofis.com/api`

## DNS
Cloudflare üzerinden CNAME/A kayıtları Vercel ve API sunucusuna yönlendirilmelidir.
